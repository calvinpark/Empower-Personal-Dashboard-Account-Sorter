# Empower Personal Dashboard Account Sorter
Firefox and Chrome extension that automatically sorts accounts on the Empower Personal Dashboard (formerly Personal Capital) by balance in descending order.

## Privacy and Security

### Zero Data Transmission
No network requests are made. Data never leaves your browser.

### Zero Storage
Account balances and credentials are not stored locally or remotely.

### Zero Tracking
The codebase contains no analytics, telemetry, or tracking scripts.
    
### Zero Leakeage
Scoped execution strictly limited to `https://ira.empower-retirement.com/dashboard/*`.
